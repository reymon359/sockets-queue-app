Queue app

An app to manage a public queue


It has 4 windows:

  • Index page: To access the other windows and create dinamically a new desk window.
  • Public window: to show the actual state of the queue of tickets and in which desk is each ticket
  • Create tickets: Where users create new tickets. Index, to access the other windows and create dinamically a new desk window.
  • Desk window: Where admins attend the queue tickets. You access it when you create a desk
